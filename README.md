# NodeJs Pre-interview Coding Test

## Run the project locally

**1.** Clone the project:

```
$ git clone https://github.com/kborel/coding-test.git
```

**2.** Test 1

- Install test 1 dependencies:

  ```
  $ cd test1
  $ npm install
  ```

- Start the app:

  ```
  $ npm start
  ```

**3.** Test 2

- Install test 2 dependencies:

  ```
  $ cd ../test2
  $ npm install
  ```

- Start the app:

  ```
  $ npm start
  ```
